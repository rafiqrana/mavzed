## Build pip installable file (whell)


```python3 -m build```


## Build snap 

```
cd <project_dir>
snapcraft
```
Install snap
```
sudo snap install <path_to_snap.snap> --dangerous
``` 
Uninstall snap

```
sudo snap uninstall mavzed


```
clean snap temp file
```
snapcraft clean
```
